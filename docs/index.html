Huellitas con Amor

Aquí en Huellitas con Amor conocemos lo tedioso que puede ser llevar a cabo un proceso de adopción de mascotas, debido a los extensos formularios y filas de espera , ya que Huellitas con Amor vela por las mascotas en estado de vulnerabilidad decidimos crear este proyecto el cual tiene la función de apoyar a las entidades especializadas en procesos de adopción de mascotas mediante un aplicativo web, que facilitara en gran medida el proceso de adopción de mascotas tanto para las entidades responsables como para las familias en busca de un amigo peludo a través de proceso agiles y eficaces, además nuestro aplicativo web cuenta con una interfaz sencilla para un navegación más cómoda en tu búsqueda de un compañero de vida y por si fuera poco Huellitas con Amor cuenta con la posibilidad de reportar una mascota extraviada, para que de esta forma te podamos ayudar a encontrar a tu mascota perdida. Por esto y más surge Huellitas con Amor y recuerda “No compres, adopta”.

Especificaciones del aplicativo

Las tecnologías que fueron usadas, fueron las siguientes:

  • Lenguaje de programación: Java script
  • Base de datos: Mysql
  • Conexión base de datos: Node Js
  • Tecnologías usadas: HTML & Css
  • Repositorio en la nube: Heroku
  • Control de versiones: Git
